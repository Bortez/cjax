Cjax

Ajax Development Framework built in PHP and JavaScript

Welcome to GitHub Pages.

Cjax is an MVC (Model-View-Controller) oriented, lightweight (>100kb) ajax framework aimed at PHP applications and websites.

Cjax is architectured in a self explanatory way, just by looking at the syntax you can tell right away what is it doing, and what's best about it is that's all there is to it; there is no Javascript to produce or inline codes in your html, just plain and simple PHP.

Cjax Lives in harmony with other frameworks and it complements PHP Frameworks such as Cakephp, CodeIgniter, Zend, and can be installed on any website or application in tops 5 minutes. It is compatible with most of all modern web browsers, and it requires zero configuration. Forget Inline Javascript, forget snipets, forget buch of JavaScript. Cjax processes your page in Event Listeners, and it is fully compatible with jQuery and with PHPFrameworks such as CodeIgniter.
